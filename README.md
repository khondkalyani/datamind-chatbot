# 🤖 DataMind AI Chatbot

## Live Demo[Try it here]
🌐(https://datamind-chatbot-wiukugq595htsarzrxfr59.streamlit.app/)

## About
A ChatGPT-like conversational AI chatbot built 
with Llama 3.3 70B model via Groq API and Streamlit.
Supports natural conversation with memory of 
previous messages in the session.

## Features
- 💬 Natural conversation like ChatGPT
- 🧠 Remembers chat history within session
- 🔑 Secure API key input — key never stored
- ➕ New chat button to start fresh
- 🗑️ Clear chat option
- ⚙️ Sidebar with settings
- 📝 Markdown and code formatting support

## Tech Stack
- Python
- Streamlit
- Groq API
- Llama 3.3 70B

## How To Use
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up for free
3. Create an API key
4. Paste key in the app
5. Start chatting!

## What It Can Do
- Answer general knowledge questions
- Write and explain code
- Explain data science concepts
- Help with statistics and math
- Have natural conversations
